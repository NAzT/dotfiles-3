# dotfiles

1. clone the repo into `~/Documents/projects/`
	```
	git clone git://repo/to/ellaqezi/dotfiles.git ~/Documents/projects/dotfiles
	```
1. ```sudo easy_install pip```
1. ```sudo pip install ansible```
